# JS homework no.6 GoIT full stack developer course - some DOM and event handlers exercises:

# Task 1. A script that: - It will count and write to the console the number of categories in ul#categories. - For each list item in the ul#categories list, it will find and print to the console the text of the item's header and the number of items in the given category.

# Task 2. A script that for each element of the ingredients array: - It will create a separate list element. Mandatory use of the document.createElement() method. - It will add the name of the element (string from the array) as its text content.- It will add the item class to the item. - Then one operation will put all list items in ul#ingredients list.

# Task 3. A script to create an image gallery by array of data. - Use an array of images objects to create image elements inside list. - To create a tag, use template strings and the insertAdjacentHTML() method.

# Task 4. The counter consists of a span element and buttons that, when clicked, should increase and decrease its value by a unit.

# Task 5. A script that, when you type text into the input field input#name-input (input event), inserts its current value into span#name-output. If the input field is empty, the text "Anonymous" should be displayed in span.

# Task 6. A script that checks whether the value entered by the user meets the validation condition (has the appropriate length) when the focus is lost on the input field (blur event).

# Task 7. A script that responds to a change in the value of the input#font-size-control element (input event) and changes the inline span#text style by updating the font-size property. As a result, when changing the value with the slider, the size of the text in the span element is supposed to change.

# Task 8. Login form management script.

# Task 10. A script for creating and deleting item collections. The user enters the number of items into input and presses the Create button, which starts the creation of the collection. After pressing the Destroy button, the collection of elements is deleted.
